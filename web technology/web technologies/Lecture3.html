Lecture 3: HTML Forms and Media

Collecting user data and adding multimedia

Summary

HTML forms are used to collect user input such as names, emails, and messages. Media elements allow developers to include images, audio, and video in web pages.

Topics Covered

  • Form Elements
  • Input Types
  • Images
  • Audio and Video Tags

Example Code

<form>
<input type="text" placeholder="Enter name">
<button>Submit</button>
</form>
