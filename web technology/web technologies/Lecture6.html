Lecture 6: CSS Box Model & Layout

Understanding spacing and layout structure

Summary

The CSS Box Model describes how elements are displayed on a webpage. It includes margin, border, padding, and content.

Topics Covered

  • Margin
  • Border
  • Padding
  • Content Area

Example Code

div {
margin:20px;
padding:10px;
border:1px solid black;
}
